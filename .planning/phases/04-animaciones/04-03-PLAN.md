BarDoorTrigger.cs (Assets/_Project/Core/SceneManagement/BarDoorTrigger.cs) — molde a copiar:
```
[RequireComponent(typeof(Collider))]
public class BarDoorTrigger : MonoBehaviour
{
    [SerializeField] private string playerTag = "player";
    private bool triggered;
    void Reset()
    {
        var col = GetComponent<Collider>();
        if (col != null) col.isTrigger = true;
    }
    void OnTriggerEnter(Collider other)
    {
        if (triggered) return;
        if (!other.CompareTag(playerTag)) return;
        triggered = true;
        // ... accion ...  <- aca BarDoorTrigger hace SceneManager.LoadSceneAsync; la puerta decorativa NO.
    }
}
```
> El tag del jugador en el proyecto es "player" (minuscula), como en BarDoorTrigger.

Asset prerequisito (Plan 02): `Assets/_Project/Animations/Door.controller` con parametro Trigger "Open", estados Closed (default) / Open. El Animator del GameObject puerta debe usar este controller como runtimeAnimatorController.
