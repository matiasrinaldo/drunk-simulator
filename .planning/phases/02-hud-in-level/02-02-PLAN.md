## Phase Goal

**As a** player, **I want to** see my drunkenness level and money at all times during the game, **so that** I can make informed decisions about when to drink more and whether I can afford the next drink.

<objective>
Crear HUDController.cs — el singleton DontDestroyOnLoad que instancia el Canvas HUD por código, muestra la barra de borrachera con lerp hacia DrunkManager.EffectIntensity, y el indicador de dinero TMP suscripto a PlayerMoneyStore.OnMoneyChanged.

Purpose: Entrega la capacidad completa de la fase — el jugador puede leer su nivel de borrachera y dinero en cualquier momento del juego, en cualquier escena y en cualquier modo de cámara.

Output: Assets/_Project/UI/HUD/HUDController.cs — script que construye el Canvas completo por código, vive como singleton entre escenas y refleja en tiempo real los dos estados de juego.
</objective>

```csharp
public class BackgroundMusicManager : MonoBehaviour
{
    private static BackgroundMusicManager instance;

    [RuntimeInitializeOnLoadMethod(RuntimeInitializeLoadType.BeforeSceneLoad)]
    private static void Bootstrap()
    {
        if (instance != null) return;
        GameObject managerObject = new GameObject(nameof(BackgroundMusicManager));
        instance = managerObject.AddComponent<BackgroundMusicManager>();
    }

    private void Awake()
    {
        if (instance != null && instance != this) { Destroy(gameObject); return; }
        instance = this;
        DontDestroyOnLoad(gameObject);
        // ... construir componentes por código (AudioSource, etc.)
    }
}
```

```csharp
public class DrunkManager : MonoBehaviour
```
